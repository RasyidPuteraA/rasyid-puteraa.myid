1. Project Overview

rasyid-puteraa.my.id adalah website personal profesional yang berfungsi sebagai pusat identitas, arsip karya, pengetahuan, dan perjalanan hidup Muhammad Rasyid Putera Agung.

Website ini tidak hanya berfungsi sebagai portfolio, tetapi berkembang menjadi arsip pengetahuan personal yang hidup (living knowledge archive) yang mendokumentasikan karya, pengalaman, pemikiran, dan pembelajaran sepanjang waktu.

Website ini juga menjadi fondasi untuk sistem eksplorasi pengetahuan bernama KOM, yang pada masa depan akan berkembang dari search system menjadi AI knowledge assistant.

2. Core Objectives

Website ini dibangun untuk mencapai beberapa tujuan utama:

Personal Identity

Menjadi representasi profesional yang jelas mengenai siapa Rasyid, bidang keahlian, dan fokus karya.

Portfolio & Projects

Menampilkan hasil karya dan project yang telah dikerjakan dalam bidang engineering, teknologi, dan pengembangan sistem.

Knowledge Archive

Menyimpan catatan pembelajaran, review buku, jurnal, artikel, dan refleksi pemikiran.

Journey Documentation

Mendokumentasikan perjalanan hidup, milestone, dan pengalaman organisasi maupun profesional.

Venture Gateway

Menjadi pintu masuk menuju venture dan platform yang dibangun oleh Rasyid (misalnya Serbada, SAKMODUL, dll).

Knowledge Exploration

Memberikan cara bagi pengunjung untuk mengeksplorasi arsip pengetahuan melalui sistem KOM.

3. Philosophy

Website ini dibangun sebagai arsip reflektif perjalanan hidup dan pengetahuan.

Isi website bukan dimaksudkan sebagai kebenaran absolut, melainkan kumpulan perspektif, pengalaman, dan refleksi yang berkembang seiring waktu.

Tujuannya adalah menjaga agar karya, pemikiran, dan pembelajaran tidak hilang, serta dapat menjadi sumber pembelajaran bagi generasi berikutnya.

4. KOM System

Website ini memiliki sistem eksplorasi pengetahuan bernama KOM.

KOM berfungsi sebagai sistem yang mengindeks dan menghubungkan seluruh konten website, termasuk:

project

artikel

catatan pembelajaran

dokumentasi

pengalaman

Makna filosofis dari KOM:

KOM — Kill of Memory

Makna ini menggambarkan proses membongkar, menguji, dan memahami kembali ingatan serta pengalaman untuk menemukan makna yang lebih dalam.

KOM bukan sekadar penyimpanan data, tetapi sistem eksplorasi pengetahuan.

5. System Evolution

Website ini dirancang untuk berkembang melalui beberapa fase:

Phase 1 — Foundation

Website portfolio dasar dengan identitas profesional dan daftar project.

Phase 2 — Knowledge Archive

Website mulai berfungsi sebagai arsip pengetahuan dan dokumentasi pembelajaran.

Phase 3 — Structured Knowledge

Konten diorganisir melalui kategori, relasi, dan struktur knowledge base.

Phase 4 — KOM Search

KOM menjadi sistem pencarian internal untuk menelusuri seluruh arsip website.

Phase 5 — KOM AI

KOM berkembang menjadi AI knowledge assistant yang dapat membantu eksplorasi dan pemahaman arsip.
